Information Collection and Use

The application will not collect personally identifiable information except for those mentioned in the sections bellow.

The application will connect to and use data provided by the game Assetto Corsa Competizione for the duration the application is running. The Data collected from the game is subject to the privacy policy of the game.

Log Data

I want to inform you that whenever you use the application, in a case of an error in the application I collect data and information (through third-party products) on your phone called Log Data. This Log Data may include information such as your device Internet Protocol (“IP”) address, device name, operating system version, the configuration of the application, the time and date of your use of the application, and other statistics. The information that is collected will be retained on your device and is not collected by me in any way.

Information collection, usage and storage when using the Google Sheet integration

When using the Google Sheets integration in ACC Race Control, the application will communicate with Google Services to provide the intended user experience. All communication with Google services is done directly from the application to Google services.

The user may connect the application to a Google Sheets Spreadsheet using the Google sheets integration of ACC Race Control. The application will then write data (collision information, race entry lists) to the spreadsheet. The application will read data from the spreadsheet to find empty fields it can write data to or to synchronize the entry list with the entry list stored in the application.
